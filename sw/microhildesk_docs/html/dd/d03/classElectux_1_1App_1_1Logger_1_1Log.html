Detailed Description

A file-based implementation of the ILog interface.

This class provides thread-safe logging to a physical file on disk. It supports log level filtering and automatic timestamping of entries.

Logs a message to the file with a timestamp and severity tag.

Formats and writes a log entry.

Parameters
messageThe string content to log.
levelThe severity of the message.
Note
This operation is thread-safe.

Processes the message by:

  1. Checking if the stream is open.
  2. Filtering based on the current log level.
  3. Generating a ISO 8601 timestamp (YYYY-MM-DDTHH:MM:SS).
  4. Appending the appropriate severity tag.
  5. Flushing the stream to ensure data is written to disk immediately.
Parameters
messageThe text content to log.
levelThe severity of the entry.
